Urgent Application

Help

If you use delegated functions your application must not be marked as urgent, as the work will be covered by the emergency certificate. You should only answer yes to the urgent question if you do not have delegated functions or there is a valid reason why delegated functions cannot be used - for example business means and there is action that urgently needs to be taken within 48 hours.

Help Note:

Only applications in respect of work which is not covered under existing Legal Aid and needs to be completed within 48 hours maybe submitted as Urgent. If this application does not meet this definiton please answer “No” to “Are you making an urgent application?”
